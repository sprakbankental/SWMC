# Swedish Word Metrics Calculations (SWMC)

The (Visual Basic.NET) project named SWMC is a .NET framework class library that contain methods for calculation of the swedish word-metrics, most of which are described in Witte & Köbler, 2019.

The (Visual Basic.NET) project ConsoleTestingApp is a test app that demonstrate very briefly how the SWMC library can be used in order to calculate word-metrics on the local computer.
